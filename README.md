🗽 New York City Airbnb Analysis Dashboard
Data Analysis & Interactive Visualization Project
📋 Project Overview
This project provides a comprehensive analysis of the Airbnb market in New York City. Using a dataset containing over 48K listings, I designed an interactive Power BI dashboard to help stakeholders (investors/travelers) understand pricing dynamics, geographical distribution, and host performance.

🛠️ Built With
Tool: Power BI Desktop

Data Source: NYC Airbnb Open Data

Techniques: Data Cleaning, DAX Measures, Data Modeling, Geospatial Analysis.

📊 Key Features & Insights
1. Executive Summary (KPIs)
Total Inventory: 48K listings managed by 11K active hosts.

Pricing Hub: Average stay cost is around $154.76, with a wide range reaching up to $10K for luxury stays.

Demand Indicator: Total reviews exceeded 227K, showing a high market velocity.

2. Market Segmentation
Room Type Distribution: A balanced mix between Entire Home/Apt, Private Rooms, and Shared Rooms.

Neighborhood Performance: Staten Island and Manhattan lead the average pricing charts, while Brooklyn and Queens offer more budget-friendly high-volume options.

3. Time Series & Trends
Growth Analysis: The dashboard tracks the Average Review per Month by Year, showing a significant upward trend in guest engagement reaching its peak in 2019.

4. Geospatial Intelligence
An interactive map pinpointing listings across NYC, allows users to visualize "Hot Zones" where listings are concentrated (Density Analysis).

🧪 Advanced DAX Formulas Used
In this project, I implemented custom DAX measures to extract deeper insights:

Occupancy Rate: Calculated based on the availability_365 column to estimate property performance.

Active Hosts: Distinct count of host_id to identify unique market players.

Performance Scoring: A composite metric balancing Price, Reviews, and Availability.
